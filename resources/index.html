    Unity

    Unity is a game engine that serves as the backbone for many VR experiences. Getting started with VR development can be as easy as dragging and dropping objects into a starter scene, with the ability to add more scripts and complexity as you learn. The Unity asset store is also a readily available source of free models, scripts, and environments to start your project off right. Unity provides a great collection of tutorials for using their engine to help you get started.

    Unity Unity Tutorials

    Unreal Engine

    Unreal Engine is an alternate video game engine to Unity. While the space focuses more on Unity development, you can use this engine for virtual reality as well. Unreal Engine provides a great deal of written and video documentation for using this software.

    Unreal Engine Unreal Engine 4 Documentation

    Blender

    Blender is a 3D computer graphics application used for game development to animation and other professional applications. The software is free to use.

    Blender

    GitHub

    GitHub is a version control platform that allows you to manage your works-in-progress from the cloud. Work by yourself or with a team, in the Immersive Space or at home. Github supports most VR development platforms available in the Immersive Space, including Unity3D and AFrame.
